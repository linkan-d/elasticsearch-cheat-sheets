# Elasticsearch Cheat Sheets

A collection of helpful cheat sheets and quick reference guides for Elasticsearch and Kibana.

## Contents

- **elasticsearch-kibana-cheat-sheet.md** - Comprehensive guide to Elasticsearch Kibana Dev Tools commands organized by category

## Categories Covered

- Cluster Health & Status
- Index Management
- Document Operations
- Search Queries
- Aggregations
- Mapping Management
- Index Aliases
- Reindex Operations
- Index Templates
- Shard Management
- Performance & Monitoring
- Snapshot & Restore
- Analysis & Tokenization
- Count & Stats
- Delete Operations
- Useful Cat APIs

## Usage

All commands can be run directly in Kibana Dev Tools console. Simply copy and paste the commands and modify the parameters as needed.
